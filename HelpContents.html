Using Sudoku Game.


Sudoku is a game that is played in a 9x9 grid with similar 3x3 grids regions in the big grid. The goal of the game is to fill the empty boxes in the grid and complete the whole puzzle. This puzzle does not require math but more‐so logic. The 3x3 grids must have the values 1‐9 only once, so duplicates would not be allowed, and the 9x9 grid in which the 3x3 grids sit, should have the values 1‐9 across horizontally and vertically, in every column and row.

Saving Files

Legandary Software Inc. developed the game of soduku with the saving capablity so that users can save the game for playing at their own time.

Opening files

You can open a saved sudoku file by going to File Menu and choosing open. You will then given a chance to locate the saved file so that you can continue playing the game.

Printing Results

In Sudoku, you have the option to print the gameboard onto a paper and solve the question. In order to print the sudoku, you must buy the full version of the game. if you have the ful version of the game, you can print by going to the File menu and choosing print or pressing the shortCuts Keys.

Progress Reports

Legendary Software Inc. offers the conveneience of checking your progress in game. As you play the game serveral times, we will calclate your progress as a graph and display the results clearly for you. To view your progress, simply go to View menu and choose Graph.. or use the shortCuts below.
